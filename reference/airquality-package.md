# airquality: Gather, Standardize, and Analyze Air Quality Data

The goal of airquality is to reduce the burden of gathering,
standardizing, and analyzing air quality data. This package is very much
a work in progress, significant changes will occur and there is no
gaurentee things will work as expected.
